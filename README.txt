Проект Windows Forms для десериализации XML и JSON.

Что сделано:
1. Программа не создает файлы, а только загружает и десериализует их.
2. В папке Data лежат два готовых файла:
   - music_festival.json
   - music_festival.xml
3. В обоих файлах одни и те же данные.
4. Тема проекта: мировой фестиваль лучших групп и артистов мира.
5. Сущности:
   - Festival
   - Artist
   - StageInfo
6. Есть две формы:
   - MainForm
   - DetailsForm
7. Есть обязательные элементы:
   - TreeView
   - DataGridView
   - SplitContainer
   - Button
   - PictureBox

Код специально упрощен:
- без лямбда-выражений
- без сокращенных приемов
- без Parse
- простые отдельные методы
- отдельные классы строк для таблицы
